## 项目介绍

* 后端采用Spring Boot、Jwt。
* 权限认证使用Jwt。

### 功能介绍
* 用户注册
* 用户登录
* 用户上传头像
* 主页搜索
* 电影详情
    * 展示评分
    * 剧照、剧情简介、演职员
* 个人页面
    * 自我介绍
    * 想看/看过
    * 我的影评
